# Proyecto_Aitana
Proyecto en Python que crea a Aitana, una amiga virtual por texto que funciona localmente con Llama 3 mediante Ollama. Incluye personalidad definida, memoria a corto plazo, memoria a largo plazo con RAG (ChromaDB/FAISS), arquitectura modular, interfaz por consola, archivos de instalación, código fuente y documentación para ejecutarlo offline.
